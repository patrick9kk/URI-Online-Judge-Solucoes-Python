# URI-Online-Judge-Solucoes-Python

> Nome atual no GitHub: `URI`. Sugestão de nome mais descritivo: **`URI-Online-Judge-Solucoes-Python`**.

Coleção de soluções em **Python 3** para problemas do [URI Online Judge](https://www.urionlinejudge.com.br/), organizadas por número do problema.

## Estrutura

Cada arquivo é nomeado com o número do problema no URI (sem extensão `.py`) e contém a solução correspondente.

## Lista de problemas resolvidos

| Problema | O que o código resolve |
|---|---|
| 1004 | Lê dois inteiros e imprime o produto entre eles. |
| 1037 | Classifica um número real em qual intervalo de 25 em 25 (entre 0 e 100) ele se encontra. |
| 1060 | Lê 6 valores e conta quantos são positivos. |
| 1073 | Imprime os números pares entre 2 e um inteiro N informado. |
| 1099 | Para cada par de números lido, soma os valores ímpares estritamente entre eles. |
| 1150 | Calcula quantos períodos de uma progressão são necessários até atingir/ultrapassar um valor lido. |
| 1168 | Conta o total de segmentos de LED necessários para exibir cada dígito de uma sequência de números. |
| 1263 | Lê linhas até EOF, agrupa as iniciais das palavras de cada linha e conta quantos grupos de letras repetidas consecutivas existem. |
| 1278 | Alinha/justifica à direita um bloco de linhas de texto, separando cada bloco de entrada por uma linha em branco. |
| 1332 | Verifica, letra a letra, se uma palavra de até 3 caracteres corresponde a "one" e classifica o resultado em 1, 2 ou 3. |
| 2717 | Verifica se a soma de dois números não excede um limite N e imprime a mensagem correspondente. |
| 2791 | Lê 4 números e imprime o índice (1 a 4) de cada um que for positivo. |
| 3037 | Compara a pontuação de dois jogadores em 3 arremessos (peso × distância) e informa o vencedor. |
| 3039 | Conta, a partir de uma lista de brinquedos, quantos são "carrinhos" e quantos são "bonecas". |
| 3053 | Simula trocas sucessivas de posição entre 3 copos (A, B, C) a partir de uma sequência de movimentos. |
| 3161 | Verifica se cada fruta (na ordem normal ou invertida) aparece em um conjunto de frases. |
| 3170 | Calcula quantas bolinhas faltam para decorar a metade dos galhos de uma árvore. |
| 3173 | Calcula diferença de tempo usando `datetime` e `timedelta`. |
| 3174 | Soma as horas trabalhadas por categoria de elfos e calcula quantos brinquedos foram produzidos, dividindo por taxas fixas de cada categoria. |
| 3208 | Verifica se um número é primo dentro de um intervalo, classificando como `BAD <divisor>` ou `GOOD`. |
| 3209 | Soma os elementos de uma lista, subtraindo 1 de cada elemento que não seja o primeiro nem o último. |
| 3214 | Simula a troca de garrafas vazias por refrigerantes cheios, contando quantos refrigerantes foram obtidos. |
| 3241 | Soma duas parcelas de uma expressão no formato `a+b`, pulando quando a entrada é exatamente `P=NP`. |
| 3299 | Verifica se um número contém a sequência "13" e classifica como "de mala suerte" ou não. |
| 3301 | Determina a ordem relativa de idade entre três pessoas (Huguinho, Zezinho e Luisinho). |

## Linguagem e recursos usados

Python 3 — `input()`, list/string slicing, f-strings, `datetime`/`timedelta`, funções auxiliares.

## Como executar

Renomeie o arquivo do problema para `.py` (ou rode diretamente, já que são scripts Python puros) e informe a entrada esperada pelo problema via stdin:

```bash
python 1004
# ou
cp 1004 1004.py && python 1004.py
```
